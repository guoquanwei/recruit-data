## ADDED Requirements

### Requirement: 招聘负责人综合看板
系统 MUST 提供招聘负责人综合看板，用于按月份查看招聘达成经营指标。

#### Scenario: 查看顶部核心指标
- **WHEN** 用户打开招聘负责人综合看板并选择月份
- **THEN** 系统展示整体目标、截止目标、实际入培数、GAP、达成率、未达成基地数、自主社招整体人效和自主社招占比

#### Scenario: 切换统计月份
- **WHEN** 用户切换年份或月份
- **THEN** 系统按所选月份刷新看板全部指标和明细

#### Scenario: 未匹配数据提醒
- **WHEN** 员工实际入培数据中存在无法匹配目标基地或渠道的记录
- **THEN** 系统 MUST 在看板中展示未匹配数据提醒

### Requirement: 全局渠道占比分析
系统 MUST 展示全局渠道目标占比、达成占比和占比 GAP。

#### Scenario: 查看渠道占比卡片
- **WHEN** 用户查看综合看板
- **THEN** 系统按渠道展示目标、截止目标、实际、目标占比、达成占比和占比 GAP

#### Scenario: 固定渠道顺序
- **WHEN** 系统展示渠道占比
- **THEN** 系统 MUST 按 `回流`、`内推`、`渠道社招`、`渠道校招`、`自主社招` 的顺序展示

### Requirement: 基地风险分析
系统 MUST 提供基地风险分析，用于识别未达成基地和主要缺口渠道。

#### Scenario: 展示基地风险列表
- **WHEN** 用户查看基地风险分析
- **THEN** 系统展示所有有目标或有实际入培数据的基地，并隐藏目标和实际均为 0 的基地

#### Scenario: 标识负 GAP 基地
- **WHEN** 基地 GAP 小于 0
- **THEN** 系统 MUST 使用风险样式标识该基地

#### Scenario: 输出未达成原因
- **WHEN** 基地未达成月度目标
- **THEN** 系统按渠道 GAP 缺口输出该基地的主要缺口渠道

### Requirement: 入培明细下钻
系统 MUST 支持从分析数字下钻查看实际入培员工明细。

#### Scenario: 查看实际入培明细
- **WHEN** 用户点击实际入培数字
- **THEN** 系统展示所属基地、工号、姓名、手机号、招聘渠道、渠道名称、入培日期、在职状态、离职日期和在职天数

#### Scenario: 明细手机号脱敏
- **WHEN** 系统展示入培员工明细
- **THEN** 系统 MUST 默认展示脱敏手机号

### Requirement: 自主社招人效分析
系统 MUST 提供自主社招人效分析，按招聘专员阶段统计参培和 7 天达成。

#### Scenario: 统计招聘专员规模
- **WHEN** 系统计算自主社招人效
- **THEN** 系统统计组织路径包含 `人才开发部` 且职位为 `招聘专员` 或 `初级招聘主管` 的人员
- **AND** 系统 MUST 将组织表招聘专员统一归一为 `base = 人才开发部`
- **AND** 系统 MUST 只统计与统计月存在在岗交集的招聘专员，排除统计月之前已离职的人员
- **AND** `初级招聘主管` 的自招达成可计入整体达成，但不作为个人招聘规模口径时应按业务配置单独控制

#### Scenario: 判断招聘专员阶段
- **WHEN** 系统计算招聘专员阶段
- **THEN** 系统以统计月最后一天是否入职满 6 个月判断正式期或试用期
- **AND** 阶段判断优先使用招聘专员 `入职日期`，缺失时才使用 `入培时间`

#### Scenario: 计算人效
- **WHEN** 系统展示自主社招人效
- **THEN** 系统按试用期、正式期和整体展示参培达成、7 天达成、招聘规模和人效
- **AND** 自主社招候选人 MUST 使用 `招聘渠道 = 自主社招` 过滤
- **AND** 候选人归属招聘专员 MUST 使用 `渠道名称` 中 `+` 前的姓名匹配招聘专员姓名，例如 `张三+JZ0001` 归属 `张三`
- **AND** `参培人效 = 参培达成 / 招聘规模`，`7天人效 = 满 7 天达成 / 招聘规模`
- **AND** 满 7 天达成口径为 `入培时间 + 7 天 <= 统计截止日` 且 7 天节点前未离职
- **AND** 综合看板和自主社招人效页的主展示口径 MUST 使用 `7天人效`，`参培人效` 作为辅助指标展示

#### Scenario: 查看人效人员明细
- **WHEN** 用户点击参培达成或 7 天达成数字
- **THEN** 系统展示对应招聘专员带来的员工达成明细
